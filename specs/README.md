# specs

本目录是项目规范的机器可读来源。详细层次与依赖关系见 [`docs/README.md`](../docs/README.md)。

| 子目录 | 内容 |
|------|------|
| `modeling-conventions.md` + `terminology.md` + `conventions.yaml` | 建模地基（A.0 冻结） |
| `modules/` | L1 模块类文本定义（6 个 YAML + config） |
| `dsl/` | L2 机构装配 DSL 语法、连接语义、案例 |
| `ir/` | IR 中间表示规范（节点/边/映射/端口附着） |
| `schema/` | JSON Schema 校验（模块 / DSL / IR 图） |
